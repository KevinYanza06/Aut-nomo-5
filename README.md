# Autonomo 5

## Descripción
Este repositorio contiene el desarrollo de la Actividad Autónoma 5, donde se analizan normativas de privacidad de datos como el GDPR y la CCPA, así como casos reales relacionados con la ética en el uso de datos: COMPAS, reconocimiento facial y publicidad algorítmica.

El trabajo incluye un análisis crítico de las normativas, reflexiones éticas y conclusiones relacionadas con la Ciencia de Datos y la protección de la información personal.

## Contenido
- Análisis de normativas de privacidad (GDPR y CCPA)
- Estudio de casos éticos:
- COMPAS
- Reconocimiento facial
- Publicidad algorítmica
- Reflexión final sobre ética y privacidad de datos
